# Resume Builder

A simple web-based resume builder that lets you:

- Fill in core resume details (name, contact, summary, skills, experience)
- Instantly preview your resume while editing
- Download the generated resume as a `.txt` file

## Run locally

Since this is a static project, open `index.html` directly in a browser, or serve the directory:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
